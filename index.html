Que 3. Perfect Number Check. Have you heard of Perfect numbers? If not let me tell you what is it, Perfect Numbers are integers that are equal to the sum of all its divisors except that number itself. Now, You are given an integer N, write a program to check whether the given number is a Perfect Number or not.

Ans:
            var Perfect_Check = (N) => 
            {
              let sum=0;
               for(let i=1;i< N;i++){
                 if(N%i===0){
                   sum=sum+i;
                 }
               }  
               if(sum==N || N==1){
                 return "YES";
               }else
               {
                 return "NO";
               }
        
               };

Que 4. Reverse a Number. Write a program which takes a numebr N as input from the user and You need to reverse the number.

Ans:  
            var Reverse_Number = (N) => 
            {
              let temp=N,rev=0;
              while(temp != 0){
                rem=temp%10;
                rev=rev*10+rem;
                temp=parseInt(temp/10);
              }
              return rev;
            
            };
